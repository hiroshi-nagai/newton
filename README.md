# newton
ニュートン法の勉強用に作成したコード。

## ◆開発環境  
 [OS]: macOS 10.14~  
 [c++] gcc version 9.2.0  
 [boost] 1.71.0  
 

## ◆ファイル説明  
/newton/main              # ビルドファイル  
/newton/newton.cpp        # ソースコード  

## ◆コード説明

#### 関数: testMethod
   近似対象の関数。この関数の実行結果の対象目的変数値(ox)に最も近づくt値(説明変数)を計算する。

#### クラス: Param
  算出対象外の目的変数を纏めたクラス

#### クラス: Newton
　ニュートン法のクラス。
  Newton::newton(計算を始める初期値=0.001, 目的変数=200.34, 関数=testMethod, 算出対象外の目的変数を纏めたインスタンス=test_param, 試行回数=1000000000, 収束誤差=0.0001);
